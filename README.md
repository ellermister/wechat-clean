## 原来这是个什么项目

`wechat-clean` 本来是个帮助用户分析微信存储空间并清理的小工具。

**它能干什么：**
- 看看微信到底把你手机的空间都用到哪去了
- 分析下那些永远删不干净的缓存文件
- 帮你理解微信是怎么"优化"你的存储空间的
- 帮你理解为什么微信自带的存储空间清理不干净
- 帮你快速清理不必要的群消息和存储空间数据，保留历史有需要的联系人数据
- 能够有效减小手机微信存储空间

**初衷：**
- 我自己手机空间不够用，微信占了17G
- 想搞清楚这些空间都被什么占了
- 并根据自己的需要写出一个简单的工具实现清理不必要的缓存和群数据
- 顺便分享给有同样困扰的朋友们
- 仅此而已

## 腾讯DMCA警告

### 被指控

根据腾讯法务发来的RTF文档, 翻译了半天才看明白，项目被指控：

1. **违反DMCA第1201条** - 绕过技术保护措施
   - 是指：我看了微信的数据库结构
   
2. **违反微信服务条款** - 逆向工程
   - 是指：我试图理解我自己手机上的数据
   
3. **威胁用户隐私和安全**
   - 是指：让用户知道自己的数据长什么样很危险

4. **侵犯知识产权**
   - 是指：你的聊天记录存储格式归腾讯所有

### 不仅如此，我这只是其中一个项目

- 这次投诉一口气举报了**30多个类似项目**
- 其中最大的项目有**4443个stars**，181个stars的也没了
- 我这个291 stars的也没得说

**投诉文件原文节选：**
> "illegally offer tools for circumventing Tencent's methods for preventing unauthorized access"

是指：**让用户访问自己的数据是非法的** 😣
